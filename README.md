# Text2img

This is **Text2img** terminal version


## Features

- It can download instagram **reels** & **posts** core code of the [@wkan_bot](https://t.me/wkan_bot)
- Prompt is not necessary for random images

- Mannual
```pip
pip install -r requirements.txt
```

```python
python wkaie.py
```

```python
python wkaie.py
```
